## 简介

博弈论的结构化思维工具。基于12个一手来源的深度调研（一手来源占比60%），提炼6个核心原理和完整的操作协议。

### 核心原理

1. **纳什均衡** - 多方决策的稳定状态
2. **占优策略** - 无论他人如何选择的最优行动
3. **零和与非零和博弈** - 竞争vs合作框架
4. **重复博弈与合作演化** - 未来阴影促进合作
5. **机制设计** - 逆向设计规则实现系统目标
6. **贝叶斯博弈与不完全信息** - 信息不对称下的策略互动

### 适用场景

- 竞争策略分析（价格战、市场份额竞争）
- 谈判与冲突（贸易谈判、薪资谈判、国际冲突）
- 机制与规则设计（拍卖设计、匹配系统、激励机制）
- 合作与演化（长期合作关系、联盟形成）

### 触发词

博弈论、game theory、纳什均衡、策略互动、零和博弈、重复博弈、机制设计
